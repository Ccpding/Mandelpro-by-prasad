## üßÆ How it Works

The Mandelbrot set is defined by the set of complex numbers $c$ for which the iterative function does not diverge to infinity:

$$z_{n+1} = z_n^2 + c$$

Where:
* $z$ starts at $0$.
* $c$ is the coordinate of the point in the complex plane.
